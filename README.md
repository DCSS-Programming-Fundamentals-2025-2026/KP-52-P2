Цей проєкт реалізує спрощену архітектуру великої мовної моделі (LLM) з можливостями навчання (Trainer) та інтерактивної генерації тексту (Chat). Проєкт розділений на дві основні команди: P1 (Trainer) та P2 (Chat).
Уніфікований інтерфейс: Створено ITextGenerator, який слугує спільним контрактом для всіх типів моделей (Mock, N-Gram, Neural).
Спільна логіка: Винесено базові сутності (Checkpoint, Sampler) у спільні бібліотеки для уникнення дублювання коду.
Парсинг команд: Реалізовано обробку команд керування параметрами безпосередньо в циклі чату:

/temp T — зміна температури випадковості.

/topk K — обмеження вибору з K найімовірніших токенів.

/seed N — встановлення фіксованого стану генератора для відтворюваності.

/reset — скидання контексту діалогу.

/help — список доступних команд.

UTF-8 Support: Повна підтримка української мови в консольному інтерфейсі.
Реалізовано завантаження моделі через аргумент -checkpoint.

Підтримка параметрів ініціалізації через командний рядок (-temp, -topk, -seed).
Проєкт побудований за модульним принципом:

Chat: Клієнтський застосунок для користувача.

Contracts: Спільні інтерфейси та контракти.

Lib.MathCore / Lib.Models: Математичні обчислення та архітектури моделей.

Lib.Tokenization: Перетворення тексту в числові токени.
ІНТЕГРАЦІЯ
Інтеграція реалізована через механізм Project References та спільні інтерфейси.

Запускати: dotnet run --project Chat -- -checkpoint model.json -temp 0.7 -topk 5
