Цей проєкт реалізує спрощену архітектуру великої мовної моделі (LLM) з можливостями навчання (Trainer) та інтерактивної генерації тексту (Chat).
Tokenization: Введений користувачем текст перетворюється у вектор числових індексів (tokens) за допомогою словника, завантаженого з чекпоінту.

Model Inference: Модель (N-Gram або Transformer) обробляє вхідні токени та видає Logits — сирі оцінки ймовірностей для кожного наступного слова зі словника.

Processing (Temperature & Top-K): Застосовується Temperature Scaling для регулювання впевненості моделі.

Виконується Top-K Filtering, щоб відсіяти малоймовірні варіанти.

Sampling: На основі фінального розподілу ймовірностей обирається наступний токен (враховуючи встановлений Seed для детермінованості).

Detokenization: Числовий індекс перетворюється назад у текст (UTF-8) і виводиться користувачу.


--------------------------------------------
/temp T — зміна температури випадковості.

/topk K — обмеження вибору з K найімовірніших токенів.

/seed N — встановлення фіксованого стану генератора для відтворюваності.

/reset — скидання контексту діалогу.

/help — список доступних команд.

UTF-8 Support: Повна підтримка української мови в консольному інтерфейсі.
